# Functional Requirements

## Core Features

- **Editable Items**: Each list item can be edited

- **Chronological Ordering**: Newly added list items should be displayed in ascending order by time

- **Hot Pink Container**: The container that includes the list of items should have a hot pink background color

- **Timestamp Display**: Include the timestamp as a subtext below the item title in "MMMM dd, yyyy" format (12-hour format with am/pm)

- **Clear All Button**: There should be a "clear all" button beside the "add item" button that can clear all of the list items at once
